# 🚀 Развертывание на Render (Бесплатно)

## Шаг 1: Подготовка GitHub репозитория

1. Создайте GitHub аккаунт (если нет)
2. Создайте новый репозиторий `yapona-mama-bot`
3. Загрузите туда файлы:
   - `main.py`
   - `requirements.txt`
   - `Procfile`
   - `.env.example`
   - `README.md`

## Шаг 2: Получение токена Telegram

1. Откройте Telegram
2. Найдите бота `@BotFather`
3. Напишите `/newbot`
4. Выберите имя и юзернейм
5. Скопируйте выданный **токен**

## Шаг 3: Регистрация на Render

1. Перейдите на https://render.com
2. Нажмите "Sign up"
3. Авторизуйтесь через GitHub

## Шаг 4: Создание Web Service

1. Нажмите "New +" → "Web Service"
2. Выберите ваш репозиторий
3. Укажите имя сервиса (например: `yapona-mama-bot`)
4. Выберите Python как окружение
5. В "Build Command" оставьте пусто или: `pip install -r requirements.txt`
6. В "Start Command" укажите: `python main.py`

## Шаг 5: Добавление переменной окружения

1. Прокрутите вниз до "Environment"
2. Нажмите "Add Environment Variable"
3. **Name:** `TELEGRAM_BOT_TOKEN`
4. **Value:** вставьте ваш токен от BotFather
5. Нажмите "Save"

## Шаг 6: Запуск

1. Нажмите "Deploy"
2. Дождитесь завершения (обычно 2-5 минут)
3. Вы увидите статус "Live"

## Шаг 7: Проверка

1. Откройте Telegram
2. Найдите вашего бота по юзернейму
3. Напишите `/start`
4. Бот должен ответить

## 📝 Важные моменты

- **Бесплатный план Render** может спать после 15 минут неактивности, бот просыпается при первом сообщении
- **БД сохраняется** в `food_bot.db` на сервере
- **Для изменений** делайте git push в репозиторий → Render автоматически пересоберет

## 🔄 Как обновить меню

1. Отредактируйте `main.py` (функция `add_default_menu()`)
2. Сделайте git commit и push
3. Render автоматически переразвернет бот
4. Удалите старую БД на сервере или сделайте `/start` заново

## 🐛 Если что-то не работает

Откройте **Logs** в Render, чтобы увидеть ошибки:
1. Нажмите на ваш сервис
2. Откройте вкладку "Logs"
3. Ищите красные ошибки
