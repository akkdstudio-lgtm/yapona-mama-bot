# 🍜 Япона Мама - Телеграм Бот для Заказа Еды

Полнофункциональный бот для заказа еды с меню, корзиной, историей заказов и оплатой.

## 🚀 Быстрый старт

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Получение токена бота
- Напишите @BotFather в Telegram
- Используйте команду `/newbot`
- Скопируйте токен

### 3. Запуск бота

**Вариант А: Локально (для тестирования)**
```bash
export TELEGRAM_BOT_TOKEN="123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11"
python main.py
```

**Вариант Б: На сервере Render (бесплатно)**

1. Зарегистрируйтесь на https://render.com
2. Создайте новый "Web Service"
3. Подключите этот репозиторий с GitHub
4. В настройках добавьте переменную окружения:
   - `TELEGRAM_BOT_TOKEN` = ваш токен
5. Запустите

## ✨ Функции

✅ **Просмотр меню** - 8 готовых блюд с описанием и ценой
✅ **Корзина** - добавление/удаление товаров
✅ **Оформление заказа** - быстрое оформление
✅ **История заказов** - просмотр всех заказов
✅ **SQLite БД** - без внешних зависимостей

## 📋 Команды бота

- `/start` - начало работы
- `/cancel` - отмена операции

## 🛠️ Структура кода

- `main.py` - основной код с логикой бота
- `food_bot.db` - база данных (создается автоматически)
- Встроенная БД: меню, заказы, корзины

## 🐛 Проблемы и решения

**Ошибка "Timeout waiting for connection"**
- Убедитесь, что токен правильный
- Интернет соединение стабильно

**БД не создается**
- Убедитесь, что у программы есть права на запись в папку
- Проверьте доступный диск

**Бот не отвечает**
- Перезагрузите приложение
- Удалите `food_bot.db` и запустите заново
